# DEVELOPMENT.md

Minimal setup and usage guide for the **cpp-pricer** component.

---

## Prerequisites

| Tool | Minimum Version | Purpose |
|------|------------------|----------|
| GCC or Clang | GCC ≥ 11 / Clang ≥ 12 | C++20 compiler |
| CMake | ≥ 3.15 | Build system |
| clang-format | — | Code formatting |
| clang-tidy | — | (Optional) Linting |

### Install on Ubuntu
```bash
sudo apt update
sudo apt install build-essential cmake clang-format clang-tidy -y
